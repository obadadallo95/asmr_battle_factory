# Onboarding Feature

## Purpose
Introduce the user to the app's concept and collect necessary setup data (API Key).

## Flow
1. **Splash Screen**: Checks `SettingsService.hasSeenOnboarding`.
2. **Onboarding Page**: 
   - Step 1: Arena (Microscopic World)
   - Step 2: Forces (Nature, Elements, etc.)
   - Step 3: Timeline (AI Generation process)
   - Step 4: Director (API Key Setup)
3. **Completion**: Sets `hasSeenOnboarding = true`, navigates to Home.

## Assets
- 4 Cinematic Images generated by AI (`assets/images/onboarding/`).
- Particle effects (`AnimatedBackground`).
