# سياسة الخصوصية

نحن في **مصنع المعارك (ASMR Battle Factory)** نؤمن بأن الخصوصية حق أساسي. تم تصميم هذا التطبيق ليعمل بشكل محلي تماماً مع احترام كامل لبياناتك.

## 1. جمع البيانات
نحن **لا نجمع** أي بيانات شخصية، صور، أو معلومات تعريفية. التطبيق يعمل بشكل مستقل على جهازك.

### مفاتيح API
- لكي يعمل التطبيق، يتطلب مفتاح API الخاص بـ Gemini (Google).
- يتم تخزين هذا المفتاح **محلياً فقط** ومشفراً باستخدام تقنيات التخزين الآمن (Secure Storage).
- لا يتم مشاركة هذا المفتاح مع أي خادم خاص بنا أو طرف ثالث.

## 2. التخزين المحلي
نستخدم تقنية **Hive** لتخزين:
- إعدادات التطبيق (اللغة، الثيم).
- تاريخ المعارك التي قمت بتوليدها.
- حالة "أول مرة استخدام".

يمكنك مسح هذه البيانات في أي وقت من خلال: **الإعدادات > مسح الذاكرة المؤقتة**.

## 3. مشاركة البيانات
لا نقوم بمشاركة أي بيانات مع جهات خارجية. الاتصال الوحيد بالإنترنت يتم مباشرة بين جهازك وبين خوادم Google (Gemini API) لتوليد النصوص.

## 4. المحتوى المولد
جميع النصوص والسيناريوهات التي يتم توليدها هي ملك لك وتخزن على جهازك.
