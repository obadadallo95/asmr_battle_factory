# حقوق الملكية الفكرية

## 1. المحتوى المولد (المستخدم)
- **ملكية كاملة:** أنت تمتلك الحقوق الكاملة لجميع النصوص والسيناريوهات التي تقوم بتوليدها باستخدام التطبيق.
- **الاستخدام التجاري:** يحق لك استخدام المخرجات لأغراض تجارية (مثل نشر فيديوهات على يوتيوب وتحقيق الدخل).

## 2. حقوق الأصوات (Foley Sounds)
- الأصوات المتوفرة داخل المكتبة الصوتية للتطبيق هي إما:
  - مرخصة تحت **Creative Commons Zero (CC0)**.
  - أو تم شراؤها بترخيص يتيح استخدامها داخل التطبيق.

## 3. الذكاء الاصطناعي
- المحتوى يتم توليده بواسطة نماذج ذكاء اصطناعي (Gemini).
- القوانين المتعلقة بحقوق النشر للمحتوى المولد بالذكاء الاصطناعي لا تزال تتطور عالمياً. ننصحك بمراجعة قوانين بلدك.

## 4. إخلاء مسؤولية
- نحن لا نتحمل مسؤولية أي انتهاك لحقوق الطبع والنشر قد يحدث نتيجة لمطابقة غير مقصودة في النصوص المولدة.
