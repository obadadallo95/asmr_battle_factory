# Gemini API Setup

## 1. Get API Key
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Create a new API Key.
3. Copy the key.

## 2. Configure App
1. Open the App.
2. Go to **Settings**.
3. Paste the key in the "Gemini API Key" field.
4. Tap **Save**.

## 3. Security
- The key is stored in `FlutterSecureStorage` (Keychain on iOS, Keystore on Android).
- It is never logged or sent to our servers.
